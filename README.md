# Credentials Manager BackEnd

## Requirements
- Postgres 15
- Jdk 17
- Maven 3

## Installation
```bash
mvn clean install
```

## Run
```bash
mvn spring-boot:run -Dspring-boot.run.profiles=dev
```

## Utility
- Swagger: http://localhost:8080/swagger-ui/index.html
